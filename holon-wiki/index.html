Holon Wiki

Algebraic approach to machine cognition — honest documentation, verified from source code.

#The Approach

Holon-lab-trading is built on algebraic primitives — bind (structure), bundle (consensus), cosine (measurement) — composed into cognitive organisms. No neural networks. No gradients. No backpropagation. Only algebra, accumulation, and accountability.

#How This Wiki Was Built

Every page was verified against source code by context-free agents (the ignorant). Each agent read Rust source or wat specs and wrote an honest transcription. Composite pages were built exclusively from verified leaf documents — no fabrication, no interpretation. The Rust source is the authority.

#Authority Chain

  1. Rust source — most honest, the living organism
  2. wat specs — the contract, catching up
  3. GUIDE / CIRCUIT / ORDER — the blueprint, catching up
  4. Website — out of date
