T0-010: Subspace — Noise Modeling and Thought Projection WHAT

T0-010: Subspace — Noise Modeling and Thought Projection

What It IS

The subspace.rs module implements online subspace learning via CCIPCA (Candid Covariance-free Incremental PCA, Weng et al. 2003). It provides three structs:

  1. OnlineSubspace — a single incremental PCA learner that tracks k unnormalized eigenvectors, a running mean, and an adaptive anomaly threshold.
  2. SubspaceSnapshot — a fully serializable record of OnlineSubspace state for persistence and distribution.
  3. StripedSubspace — N independent OnlineSubspace instances (one per "stripe") for crosstalk-free attribution with root-sum-of-squares (RSS) aggregate residual.

The module also provides a standalone jacobi_eigenvalues() function for computing eigenvalues of small symmetric matrices via Jacobi rotations.

"Inputs must be &[f64]. Callers convert bipolar Vector with vec.to_f64() before passing in — this keeps the memory layer decoupled from the VSA vector type."
— OnlineSubspace struct doc, line 63–65

How It Works

OnlineSubspace

Core State

The component matrix is stored as ndarray::Array2 so that "dot products, projections, and deflations compile down to vectorized (SIMD) loops."

CCIPCA Update (update())

On each observation x:

  1. Compute residual before updating — "ensuring the adaptive threshold is calibrated against the same distribution as test-time scoring."
  2. Update running mean: mean = ((n-1)/n) mean + (1/n) x
  3. Center: x_c = x - mean
  4. First observation: set component 0 to x_c, mark initialized.
  5. For each component i (CCIPCA step):
  • Apply decay+grow: component[i] = ((n-1-amnesia)/n) component[i] + ((1+amnesia)/n) x_c_proj * x_c
  • Deflate x_c by subtracting its projection onto the updated component, so each component captures successively smaller variance.
  1. Update threshold EMA (with warmup: uses 1/n instead of ema_alpha until n > 1/alpha).
  2. Reflexive dimensionality: maybe_grow() or maybe_shrink() (mutually exclusive per update).
  3. Periodic re-orthogonalization if reorth_interval > 0 and n % reorth_interval == 0.

Scoring (residual())

Centers the input, then iteratively projects onto each component and deflates. Returns the L2 norm of the remaining (out-of-subspace) residual — the anomaly score. Higher = more anomalous.

"Returns the residual norm (anomaly score). Higher = more anomalous."
— residual() doc, line 323

Adaptive Threshold (threshold())

"Adaptive anomaly threshold: EMA(residual) + sigma_mult * sqrt(variance)."

Returns f64::INFINITY until at least 2 observations have been seen.

Projection & Reconstruction

  • project(x) → k coefficients: the projection onto each unit-normalized component.
  • anomalous_component(x) → x - reconstruct(x): the out-of-subspace part.

Approximate Eigenvalues

"Approximate eigenvalues: L2 norms of the unnormalized component vectors."

The L2 norms of the unnormalized component vectors approximate the eigenvalues of the covariance matrix.

Explained Variance Ratio

"Estimated from recent residuals. Only meaningful after ≥10 updates."

Formula: (1.0 - res_ema² / dim).max(0.0).

Subspace Alignment (subspace_alignment())

Measures directional similarity between two subspaces via principal angles:

  1. Extract active (non-zero-norm) unit basis vectors from both subspaces.
  2. Compute M = U · Vᵀ.
  3. Compute eigenvalues of MᵀM via jacobi_eigenvalues().
  4. Take square roots → cosines of principal angles.
  5. Average the top angles (default: max(3, n/4)).
"Returns a value in [0, 1]: 1.0 = same directions, 0.0 = orthogonal."
"Cost: O(k · dim) for the basis product, O(k²) for the SVD."

Reflexive Dimensionality (Grow / Shrink K)

The subspace automatically adjusts its rank based on eigenvalue energy ratios:

  • Mutually exclusive per update: "a newly grown component starts at zero and would be immediately shrunk if both ran in the same step."
  • Defaults: min_components = initial_k, max_components = max(64, 8*k).

Re-orthogonalization

Modified Gram-Schmidt preserving component norms (eigenvalue estimates):

"Modified Gram-Schmidt re-orthogonalization preserving component norms."

For each component i, subtract projections onto all prior components, then rescale to restore the original norm.

StripedSubspace

"N independent OnlineSubspaces, one per stripe." Each stripe learns and scores its own portion of encoded data.

  • Aggregate residual: RSS of per-stripe residuals — "preserving the geometric interpretation of anomaly magnitude."
  • Residual profile: per-stripe residual vector — "the directional signal — the pattern of which stripes are anomalous — complementing the scalar magnitude (RSS)."
  • Threshold: RSS of per-stripe thresholds.
  • Used with Encoder::encode_walkable_striped which "distributes leaf bindings across stripes via FQDN path hashing."

Jacobi Eigenvalues

Standalone function for small symmetric matrices (typically k×k where k = 3–128):

  • O(k³) convergence, tolerance 1e-12, max iterations 100×k².
  • Classic Jacobi rotation: find largest off-diagonal element, compute rotation angle, apply 2×2 rotation to rows/columns.
  • Returns eigenvalues sorted descending.
